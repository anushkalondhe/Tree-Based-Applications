## Overview
This repository contains a case study on Discrete Mathematics focusing on practical applications.

## Topics Covered
- Tic Tac Toe Game  
- File Directory Structure  
- Algebraic Expression Tree  

## Description
This case study demonstrates how discrete mathematics concepts like trees, graphs, and logical structures are applied in real-world scenarios such as game logic, file organization, and expression evaluation.
